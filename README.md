# Better Shorts 🎬

**Better Shorts** is a Chrome extension designed to improve and stabilize the YouTube Shorts viewing experience. 

## Features

- 🌀 **Scroll Snap Stabilization**  
  Keeps the active YouTube Short locked and centered in the viewport when resizing the browser window, preventing YouTube's default behavior of losing track of the video.

- ⏸️ **Auto Pause & Resume**  
  Automatically pauses the playing Short when you switch to another browser tab or minimize the window. It automatically resumes playback when you return (respecting manual pauses, so it won't force-play a video you manually stopped).

- 🎨 **Premium Aesthetic**  
  A modern, compact popup themed in a deep purple glassmorphic style for quick, intuitive toggling of settings.

---

## Installation

1. Clone or download this repository.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** (top-left button) and select the `better-shorts` folder.
